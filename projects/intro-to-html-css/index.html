What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages and applications. HTML structures the content on the web.

Basic HTML Example

<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, world!</h1>
    <p>This is my first webpage.</p>
</body>
</html>

What is CSS?

CSS stands for Cascading Style Sheets. It is used to control the style and layout of web pages, allowing you to make your HTML look visually appealing.

Basic CSS Example

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    color: #333;
}
h1 {
    color: #0056b3;
}
p {
    font-size: 16px;
}
